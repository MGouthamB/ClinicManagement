API Details

Add Patient API

This API allows external systems to create a new patient in the database. No affiliation is required to create the patient.

Method : POST

URL : "/api/add_patient/"

Example Request (JSON):

{
  "name": "Example Name",
  "address": "Example address",
  "phone_number": "000-000-0000",
  "date_of_birth": "0000-00-00",
  "last_4_ssn": "000",
  "gender": "M"/"F"
}

View Add Patient API »

Add Doctor API

This API allows external systems to add a new doctor to the system. The doctor can be added without affiliation to a clinic or patients.

Method : POST

URL : "/api/add_doctor/"

Example Request (JSON):

{
  "npi": "0000000000",
  "name": "Dr. Example",
  "email": "example@brightsmile.com",
  "phone_number": "000-000-0000",
  "specialties": ["Cleaning", "crown"]
}

View Add Doctor API »

Get Clinic Details API

This API allows external systems to retrieve details of a specific clinic without showing affiliated patients and doctors.

Method : GET

URL : "/api/get_clinic/1/"

Example Response (JSON):

{
  "name": "Example Clinic",
  "phone_number": "000-000-0000",
  "city": "Fairfax",
  "state": "VA",
  "email": "contact@brightsmile.com"
}

View Clinic Details API »
